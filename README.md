# Media Catalog Application

A local-only media cataloging app that tracks books, movies, games, etc. with search and filter functionality.

## Features

- **Add/View/Delete Items**: Easily manage your media collection
- **Search and Filter**: Find items by title, category, or status
- **Sorting**: Sort items by title, category, status, rating, or creation date
- **Status Toggle**: Mark items as watched/read/unwatched
- **Export to CSV**: Export filtered lists to CSV files
- **Image Support**: Add cover/poster image paths to items
- **Statistics**: View total items, category breakdown, and status distribution
- **User-friendly GUI**: Built with tkinter for a clean, intuitive interface
